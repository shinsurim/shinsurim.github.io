  • 신수림(팀장) - 2차시 활동

  • 박연진 - 2차시 활동
  • 최준섭 - 2차시 활동

오늘의 목표

  1. 다이브 gitbub clone 하기
  2. 다이브 메인 페이지 소개에 Youtube 아이콘 추가하기
  3. 다이브 gitbub Fork 후 Pull requests 하는 법 배우기

오늘의 결과

  1. clone 후 간단한 코드 설명(css, res, footer.html 등) 후 이해함
  2. 기존에 있던 다른 아이콘과 관련된 코드를 참고하여 새로운 아이콘인 Youtube 아이콘을 만듦
    1. 아이콘 svg를 찾아 사용함
    2. 크기, 간격을 조금씩 수정하며 적절한 위치를 찾음
  3. 수정한 코드를 pull requests하여 동아리 선배가 수정한 내용을 확인할 수 있도록 함
  4. 페이지에서 수정할 부분 확인
    1. 연도 별로 활동, 사진 정리하기
    2. 부서 별로 임원들 확인할 수 있도록 하기(예쁘게)

4차시 활동

참석자

  • 신수림(팀장) - 4차시 활동
  • 권민서 - 4차시 활동
  • 박연진 - 4차시 활동
  • 최준섭 - 4차시 활동

오늘의 목표

  1. 컴퓨터 비전 공부
  2. 이미지 생성 모델을 만들기 위해 데이터 전처리 방법 학습.

오늘의 결과

  1. 이미지 데이터 전처리 과정
  • 모델 입력 크기가 고정되어 있어 모든 이미지를 동일한 크기로 리사이징해야 함
  • 학습 안정성을 높이기 위해 픽셀 값을 [0, 1] 또는 [-1, 1] 범위로 정규화함
  1. 모델의 일반화 능력 향상
  • 회전, 이동, 크롭, 플립, 노이즈 추가 등의 데이터 증강 기법을 적용해 데이터셋의 다양성을 높임
  1. 이미지 분류 모델 학습
  • 레이블을 바이케이코드(One-Hot Encoding) 형태로 변환하여 모델의 출력과 비교함
  1. CNN의 한계와 해결책
  • CNN의 깊이가 깊어질수록 그라디언트 소실 및 폭발 문제가 발생할 수 있음
  • 이를 해결하기 위해 ResNet 구조를 사용함
  • ResNet은 블록 단위로 이전 값을 더하는 스킵 연결 방식을 통해 네트워크 깊어져도 성능을 유지함

5차시 활동

참석자

  • 신수림(팀장) - 5차시 활동
  • 권민서 - 5차시 활동
  • 박연진 - 5차시 활동
  • 최준섭 - 5차시 활동

오늘의 목표

  1. 4차시 활동 내용에서 추가하여 컴퓨터 비전 심화 공부
  2. GAN모델을 참고하여 학습 및 생성 방식을 설계

오늘의 결과

  1. 발치수 예측 모델에서 발생하는 문제
  • 스케일 문제 해결을 위해 이미지에 표준 크기의 참조 객체(신용카드, 운전면허증 등)를 포함시킴
  • Segmentation을 통해 발자국과 신용카드를 분리함
  • Mask R-CNN을 사용하여 각 객체의 마스크를 예측함
  1. 발자국 크기 측정 과정
  • 마스크 객체에 대해 회전수정을 위해 PCA로 일관된 방향으로 정규화함
  • 회전수정된 마스크를 통해 발자국 크기를 픽셀 단위로 측정함
  1. 이미지 생성 모델의 다양한 종류
  • 오토인코더: 데이터를 압축하고 복원하는 방식으로 학습함
  • VAE: 확률적 모델링을 통해 다양한 이미지를 생성함
  • GAN: 생성자와 판별자가 경쟁하여 사실적인 이미지를 생성함
  • PixelRNN, PixelCNN: 픽셀 단위로 이미지를 생성함
  • StyleGAN: 이미지 스타일과 내용을 분리하여 제어할 수 있음
  1. 모델 설계
  • GAN을 채택하여 모델을 설계할 예정임

6차시 활동

참석자

  • 신수림(팀장) - 6차시 활동
  • 권민서 - 6차시 활동
  • 박연진 - 6차시 활동
  • 최준섭 - 6차시 활동

오늘의 목표

  1. 강화학습 공부
  2. 이미지 생성 모델을 만들기 위해 데이터 전처리 방법 학습
  3. GAN모델을 참고하여 학습 및 생성 방식 설계

오늘의 결과

  1. GAN의 주요 문제점
  • 훈련 불안정성: 생성자와 판별자의 학습률을 다르게 설정하고, 가중치 초기화와 작은 학습률 사용으로 조절
  • 모드 붕괴: 미니배치 디스크리미네이션, 경험 재생, 분해기술 등을 통해 생성 이미지의 다양성 유지
  • 판별자의 과도한 강력함: 판별자의 네트워크 구조 단순화, 학습률 낮춤, 학습 간격 조정으로 해결
  1. Hyperparameter 튜닝
  • 적절한 학습률, 배치 크기, 잠재 공간 크기 실험적으로 최적화
  1. 데이터 전처리 및 증강
  • 데이터 정규화와 증강을 통해 학습 데이터 품질과 다양성 확보
  1. 손실 함수 선택
  • WGAN, LSGAN 사용으로 학습 안정성과 이미지 품질 향상
  1. 모델 아키텍처 선택
  • DCGAN, Residual Blocks 등을 활용해 GAN 성능 최적화
  1. 강화학습 개요
  • 에이전트가 환경과 상호작용하면서 최대 보상을 얻기 위해 행동을 학습
  1. 강화학습의 주요 요소
  • 에이전트: 학습을 진행하는 주체
  • 환경: 에이전트가 상호작용하는 세계
  • 보상: 에이전트의 행동에 대한 피드백
  1. 강화학습 과정
  • 에이전트가 행동을 선택하고, 환경에서 보상을 받아 학습 조정
  1. 강화학습 종류
  • 모델 기반: 환경의 모델을 예측하여 최적 정책 찾기
  • 모델 프리: 환경 모델 없이 시뮬레이션으로 최적 정책 찾기 (예: Q-learning, SARSA)

팀 회고

신수림: 목표를 세우고 그 목표에 대해 하나부터 열까지 스스로 찾아보고, 선배에게 물어보거나, 팀원들과 이야기하며 서로 알려주고 부족한 부분을 채워갈 수 있었던 시간이었다. 능동적으로 하는 것이 익숙하지 않아 처음에는 힘들었지만 이번 기회를 통해 능동적으로 하는 법을 얻을 수 있었던 기회였다.

권민서: 모각코 활동을 통해 집중력을 유지하며 인공지능을 꾸준히 공부할 수 있었다. 또한 함께 공부했던 친구들 덕분에 더 열심히 공부를 할 수 있었다.

박연진: 다같이 코딩하면서 각자 까먹었던 코드나 개념들을 서로 보완해줄 수 있어서 코딩실력향상에 많은 도움이 됐다. 그리고 경험이 많은 선배와 함께 작업하면서 배우는 속도가 빨라지고 다른 사람들의 코딩 스타일도 참고하여 여러방면에서 배울 점이 많아진 점에서 유익하고 좋았다.

최준섭: 이번에 기회되서 다이브 기획부 인턴을 맡게 되었습니다. 각종 대회 준비와 홈페이지를 수정하는 작업등경험해보지 못한 일들을 이번에 팀원과 같이 활동하며 배우게 되어서 뜻 깊었던 시간이었습니다.특히 전공 수업에서 다루지 않지만 관련 분야에서 꼭 필요한 것들을 이번에 처음 접하게 되어서 더욱 가치있는 시간이었습니다.
